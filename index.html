สวัสดีครับ, ผมนายอภิรมณ์ พุ่มพวง

นักศึกษาคณะวิทยาศาสตร์และเทคโนโลยี สาขาวิทยาการคอมพิวเตอร์

มหาวิทยาลัยหัวเฉียวเฉลิมพระเกียรติ

เกี่ยวกับผม

ผมเป็นนักศึกษาวิทยาการคอมพิวเตอร์ที่มีความชำนาญในด้านการเขียนโปรแกรมHTML Javascript Python และPHPตลอด3ปีที่เรียนมาและอยากพัฒนาการเขียน React.js และ รูปแบบอื่นๆเพื่อนำไปต่อยอดกับสายงานในอตาคต

โปรเจคที่เคยมีส่วนร่วมตอนเรียน

  • Game 2D:Fake street fighter (พฤศจิกายน 2023)

    - พัฒนาเกมนี้ด้วย HTML โดยใช้ Canvas เป็นโครง
    - เขียนสคริปต์ JavaScript สำหรับการควบคุมการเคลื่อนที่ของตัวละคร ระยะเวลาแต่ละแมตช์ Hitbox และการสรุปผล

  • PHP:Mah Express (พฤษภาคม 2024)

    - ออกแบบและพัฒนาหน้าเว็บด้วย HTML โดยมี Bootstrap ช่วยตกแต่ง
    - ใช้ PhpMyAdmin ในการจัดสรร Database และ HTML ประกอบในการเชื่อมต่อกับฐานข้อมูล

  • Python:Malware Detection (พฤศจิกายน 2025)

    - ออกแบบและพัฒนาหน้าเว็บด้วย HTML
    - ใช้ Python เป็นตัวหลักในการทำงานและใช้ Discord ในการแจ้งเตือนว่าไฟล์นั้นมี Malware หรือไม่

  • ESP32:Reactor Tester (เมษายน 2025)

    - ออกแบบและพัฒนาด้วยภาษา C++
    - ใช้ I2C แสดงจอ I2C และ ESP-NOW ด้วย Serial protocol กับ UART ในการเชื่อมต่อให้บอร์ดทั้ง 2 ตัวสามารถสื่อสารกันได้
